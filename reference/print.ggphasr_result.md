# Print method for gg_phase_plane() output

Prints the plot and, if present, a summary of the classified equilibria.

## Usage

``` r
# S3 method for class 'ggphasr_result'
print(x, ...)
```

## Arguments

- x:

  A `ggphasr_result` object returned by
  [`gg_phase_plane()`](https://jmgraham30.github.io/ggphasr/reference/gg_phase_plane.md).

- ...:

  Ignored.

## Value

Invisibly returns `x`.
